A collection of BASH scripts I use.

Read and understand any of these scripts you intend to use before using them!

Scripts:

aur_search - Go to the AUR package page for whatever is in your clipboard. (If it exists) (swap out vivaldi if you use a different browser)

define - Gives you the definition of a word you have in your clipboard.

temp_conv - convert from Farenheit to Celcius, or vice versa.

wikipedia - Go to a Wikipedia article for whatever is in your clipboard (If it exists) (swap out vivaldi if you use a different browser)
